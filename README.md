# $FLOWER Generator

$FLOWER Generator is themed as a machine-side harvester program that works for a remote server and accumulates FLOWER over time.

## Project Theme

This project presents the idea of a client program that:

1. Runs on your machine.
2. Performs background work requested by the server.
3. Builds FLOWER rewards as uptime and workload increase.
4. Locks into a focused full-screen run sequence once started.

## Experience Flow

1. User lands on a branded $FLOWER Generator start screen.
2. User presses Start to begin the harvester session.
3. Video enters fullscreen and starts with sound after user interaction.
4. Countdown appears at the top as: until you can escape.
